# Access Control

## Contributors

**Author:** [Daniel Feller](https://twitter.com/djfeller)

The Access Control cloud service provides the following capabilities to SaaS and web apps:

-  SSO to SaaS apps
-  Enhanced SaaS app security
-  URL filtering within SaaS apps
